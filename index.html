My Projects

Album Art App


This project is a group assignment created in my Web Programing Class at Humber College. Album Art App is an interactive website that lets users create parody album covers featuring themselves. It is designed similarly to a photobooth (a countdown, capture, and then final result at the end). The Gallery page acts as a personal collection of the user's creations where the user can download the image, copy the image, share the image using the Web Share API on mobile devices and/or delete the image from local storage.

Web Storage Project


This assignment was created in my Web Programing Class at Humber College. I created a simple web page that allows users to input their name, save it locally using the HTML5 Web Storage API, and retrieve it whenever they visit the page. I also added a feature to display a welcome message based on the time of day.
